Competition Unit 2 Recovery Project source code of the Advanced Databases subject. This product consists of a simple clothing store, with three basic modules: shopping, inventory, and sales. Further details are described in the project requirements document (Spanish). 

This project was built with the following technologies:

**Front-End:**
- Java Swing

**Back-End:**
- Java SE with Maven as dependency manager
- JPA Framework

**Database:**
- MySQL

This project also follows a Layer-Based Architecture:

- Presentation Layer
- Business Logic Layer
- Data Access Layer
- Domain Layer
